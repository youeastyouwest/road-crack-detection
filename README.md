# 全流程道路裂缝智能检测系统

## 📋 项目概述

一款基于无人机视频流的自动化道路裂缝识别与量化评估系统。  
实现从数据采集、实时分析、历史管理到智能报告生成的全流程闭环。

## 🏗️ 系统架构

```
┌─────────────┐    ┌──────────────┐    ┌────────────────┐
│  无人机视频流  │ → │  后端服务     │ → │  裂缝检测模型   │
│  (RTMP/RTSP)  │    │  (API+WS)    │    │  (AI推理引擎)   │
└─────────────┘    └──────────────┘    └────────────────┘
                        ↕                       ↕
┌─────────────┐    ┌──────────────┐    ┌────────────────┐
│  前端仪表盘   │ ← │  Agent报告    │ ← │  量化评估引擎   │
│  (实时/历史)  │    │  (自动生成)    │    │  (裂缝分析)    │
└─────────────┘    └──────────────┘    └────────────────┘
```

## 🚀 技术栈

- **前端**: React / Vue + TypeScript + ECharts
- **后端**: Python (FastAPI) + WebSocket
- **数据处理**: PyTorch + OpenCV + ONNX
- **数据库**: PostgreSQL / InfluxDB（时序数据）
- **部署**: Docker + Nginx
